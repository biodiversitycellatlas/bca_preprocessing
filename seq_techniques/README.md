For each sequencing method, the barcode files have been renamed and forced to follow
a specific file name structure, to avoid redundancy in the code. 

The original barcode file names were: 

- Parse Biosciences:
    barcodes_R1.txt : bc_data_n26_R1_v3_4
    barcodes_R2.txt : bc_data_v1
    barcodes_R3.txt : bc_data_R3_v3

- BD Rhapsody:
    barcodes_R1.txt : BD_CLS1.txt
    barcodes_R2.txt : BD_CLS2.txt 
    barcodes_R3.txt : BD_CLS3.txt
    
    
- OAK-seq:
    barcodes_R1.txt : i5 indices (experiment)
    barcodes_R2.txt : 3M-february-2018.txt
