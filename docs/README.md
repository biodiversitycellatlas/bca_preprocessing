# biodiversitycellatlas/bca_preprocessing: Documentation

The biodiversitycellatlas/bca_preprocessing documentation is split into the following pages:

- [Usage](usage.md)
  - An overview of how the pipeline works, how to run it and a description of all of the different command-line flags.

- [Configuration Parameters](CONFIGURATION_PARAMETERS.md)
  - An overview of the different parameters a user can use within their configuration files.

- [Installation External Pipelines](INSTALLATION_EXTERNAL_PIPELINES.md)
  - Instructions on how to download and prepare external pipelines which are run simultaniously with the biodiversitycellatlas/bca_preprocessing pipeline.

- [Output](output.md)
  - An overview of the different results produced by the pipeline and how to interpret them.
