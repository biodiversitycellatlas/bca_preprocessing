# biodiversitycellatlas/bca_preprocessing: Documentation

The biodiversitycellatlas/bca_preprocessing documentation is split into the following pages:

- [Usage](usage.md)
  - Explenation of how the pipeline works, how to run it and a description of the command-line flags.

- [Configuration Parameters](CONFIGURATION_PARAMETERS.md)
  - An overview of the different parameters a user can use within their configuration files.

- [Installation External Pipelines](INSTALLATION_EXTERNAL_PIPELINES.md)
  - Instructions on how to download and prepare external pipelines which are (optionally) run simultaniously with the biodiversitycellatlas/bca_preprocessing pipeline for validation purposes.

- [Output](output.md)
  - An overview of the various files & results produced by the pipeline and how to interpret them.
